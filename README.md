# Pixel-Peak

## Overview

Pixel-Peak is a video/photo browser application made with the MERN stack. It utilizes the Pexels API for video/photo retrieval, providing users with a seamless browsing experience for discovering and exploring multimedia content.

## Features

- **Video/Photo Browsing**: Users can browse through a vast collection of videos and photos fetched from the Pexels API.
- **Search Functionality**: The application offers search functionality, allowing users to find specific videos or photos based on keywords.
- **MERN Stack**: Built with MongoDB, Express.js, React.js, and Node.js, Pixel-Peak provides a robust and efficient web application framework.
- **Responsive Design**: The user interface is designed to be responsive, ensuring a consistent experience across various devices and screen sizes.
